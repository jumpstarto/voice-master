VoiceMaster Bot - Advanced Version

Files included:
- bot.py : main bot
- keep_alive.py : keep bot online on Replit/UptimeRobot
- database.db : SQLite DB
- README.txt : instructions

Instructions:
1. Upload all files to Replit
2. Set environment variable TOKEN with your bot token
3. Run bot.py
4. Use UptimeRobot to ping the Replit URL every 5 min to keep it alive
